== Description ==

This plugin shows your events and your ticketing information into your wordpress page. 

* Connects automatically notikumi and your wordpress page
* Totally transparent
* Custom design (html markup, css and urls)
* Leveraging your SEO, you will have a complete shareable page per event.
* Checkout process integration automatically.


== Installation ==

1. Install the plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the notikumi admin page, go to "Contenido"
4. Choose your content (can be one, one or more events)
5. It will show you a shortcode like: [notikumi id="1"]
6. Go to the Page or Post you want to show the events content and paste the shortcode (in wordpress 5 create a shortcode block instead).
7. The plugin automatically connects your account
